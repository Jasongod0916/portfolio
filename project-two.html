專案 02

AI Drone-assisted Agriculture Job Marketplace

Drone + Agri Platform

一套結合無人機巡檢、YOLO 視覺辨識與 Web 平台協作的農業媒合系統，聚焦於解決農業缺工、巡田效率不足與任務協調不易的問題。系統讓農民能透過平台建立任務、安排田區與工作流程，並結合 DJI Tello 與即時影像回傳進行農地觀測，提升農務媒合與現場巡檢的效率。

展示影片

功能

  • 無人機輔助農地巡檢
  • 農業工作媒合平台
  • 任務排程與田區設定
  • 即時影像回傳監控
  • 農民與青年協作流程

技術

  • Python
  • YOLO
  • DJI Tello
  • Computer Vision
  • MySQL
  • Web 平台

成果

  • 2023 InnoServe Competition 無人機應用組第一名
  • 以無人機任務支援回應農業人力短缺問題
  • 透過數位化協調提升農地巡檢效率
